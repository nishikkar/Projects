# CS50 Projects Collection

A repository containing my solutions and projects from Harvard's CS50 courses.

## 📚 About

This repository contains my implementations of various projects and problem sets from the CS50 curriculum, including CS50x, CS50AI, and other related courses. These projects demonstrate my learning journey through computer science fundamentals.

## 🗂️ Projects Included

### **Web Application (PROJECT)**
A full-stack web application built with Flask, HTML, CSS, and JavaScript.
- **Technologies**: Python, Flask, SQLite, HTML/CSS, JavaScript
- **Features**: User authentication, dashboard, order management system
- **Structure**:
  - `app.py` - Main Flask application
  - `helpers.py` - Utility functions
  - `templates/` - HTML templates
  - `static/` - CSS and JavaScript files
  - `project.db` - SQLite database

### **Nim Game (nim)**
Implementation of the mathematical strategy game Nim.
- **Technologies**: Python
- **Files**:
  - `nim.py` - Core game logic
  - `play.py` - Interactive game interface
- **Features**: AI opponent, customizable game settings

### **Sentence Parser (parser)**
A natural language processing tool for analyzing and parsing sentences.
- **Technologies**: Python
- **Files**:
  - `parser.py` - Main parsing logic
  - `requirements.txt` - Python dependencies
  - `sentences/` - Sample text files for parsing
- **Features**: Text analysis, sentence structure detection

### **Shopping List Analysis (shopping)**
A data analysis program for shopping patterns.
- **Technologies**: Python, CSV processing
- **Files**:
  - `shopping.py` - Analysis script
  - `shopping.csv` - Sample shopping data
- **Features**: Data parsing, statistical analysis

### **Traffic Analysis (traffic)**
Computer vision project for traffic sign recognition.
- **Technologies**: Python, TensorFlow/Keras
- **Files**:
  - `traffic.py` - Neural network implementation
  - `requirements.txt` - Dependencies
- **Features**: Image classification, neural network training

## 🛠️ Technologies Used

- **Languages**: Python, HTML, CSS, JavaScript, SQL
- **Frameworks**: Flask
- **Tools**: Git, SQLite, CSV processing
- **Libraries**: Various Python libraries for AI/ML, data analysis, and web development
